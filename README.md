# CompactScrollView
ScrollView learnt to wrap its content! 😮‍💨

### How to use

The scroll view's size is adjusted according to its content:

```swift
CompactScrollView {
    Rectangle()
        .fill(.green)
        .frame(width: 100, height: toggle ? 100 : 1000)
}
```

For details see the Example app.
